Research Application Workflow

System Architecture & Data Flow

Phase 1: Discovery & Filtering

Tool: python -m jobhunt.cli discover

Input: preferences.yaml (Keywords like "Research", "Scientific")

Output: data/jobs.xlsx updated with NEW jobs.

↓

Phase 2: Hybrid Scoring Gate

Threshold Check: score >= threshold (e.g., 35%)

High Score: Auto-Tailor
Low Score: Ask User to Skip/Discovery
↓

Phase 3: Tailoring (LLM)

Skill: resume-tailoring (Tailors CV Content)

NEW: Cover Letter Generator (Custom for Research context)

Output: outbox/ folder with (MD, DOCX, CV, CL)

↓

Phase 4: Approval & Dispatch

Manual: You review the outbox/ files.

Command: python -m jobhunt.cli send
